1. Design System
Avoid:  
Custom shadows on every element  
10 different font weights  
Random paddings that don't scale  
Overdesigned icons that slow loading  

Instead:  
Pick one consistent radius  
One text color family  
One base spacing unit (multiply by 1x 2x etc)  
One icon set you actually like

2. Intentional flows
> Add detailed views that expand naturally
> Every tap should feel intentional. 
> Main screen shows overview.  Detail screen shows depth

3/ Micro Animations
> Layer in micro-interactions
> Day progress bars. Edit states. Smooth transitions. 
> These details separate good apps from great ones.
